- Python versão 3.7
- `numpy` versão 1.5 ou mais
- `scipy` versão 0.10 ou mais
- `matplotlib` versão 1.3 ou mais
- `scikit-learn` versão 0.14 ou mais
- `ipython` versão 2.0 ou mais
- `seaborn` versão 0.5 ou mais
